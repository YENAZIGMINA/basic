표 만들기

html에서 표를 구현할 때 사용하는 태그

  • tr(table row):행
  • td(table data):열(행의 내용)
  • th(table heading):행,열의 제목(굵은서체, 가운데정렬 기본값)
  • 속성 rowspan="3" : 3칸 행 병합
  • 속성 colspan="2" : 2칸 열 병합
